Projects

Gender Detection

[Link to the Project]
2018 - Present

This project is dedicated towards making foray into the field of speech processing with a Python Implementation of Gender Detection from Speech. This is done by making use of Mel Frequency Cepstral Coefficients (MFCCs). These Coefficients are then passed onto Gaussian Mixture Models (GMMs) inorder to make a prediction on the gender Detection.

Details:

The procedure used here for Gender Detection comprises of 4 steps:

  • Working with speech frames
  • Extracting MFCC features
  • Training gender models using GMMs
  • Evaluating performance on subset of Google's AudioSet corpus

Technologies used:

  • Numpy
  • Scikit-learn
  • GMM
  • MFCC
